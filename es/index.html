Cajas decodificadoras

Este trabajo fue solicitado por el dueño de un negocio privado para ofrecer un servicio de alquiler online de cajas decodificadoras de teléfonos móviles.

El propietario quería un trabajo rápido con pocas implementaciones, así como una interfaz predefinida por él.

El trabajo se completó en el plazo especificado de 7 días, pero pasado ese plazo quería añadir otras funcionalidades.

Al ser un proyecto de corta duración, hubo un problema en el código al mostrar el nombre del usuario que inició sesión, el cual se resolvió con una iteración ya que la 3ra vez que se solicitó el dato devolvió la cadena correcta.

La obra se completó satisfactoriamente.


Aquí se utilizaron NodeJS, MongoDB, Express.
